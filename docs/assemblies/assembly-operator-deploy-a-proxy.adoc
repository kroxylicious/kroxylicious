= Deploying a proxy

[role="_abstract"]
Deploy a basic proxy instance with a single virtual cluster exposed to Kafka clients on the same Kubernetes cluster.

== Prerequisites

* The operator must be installed in the Kubernetes cluster
* A Kafka cluster to be proxied
* For some ingress configurations using TLS, you will need the means to generate TLS server certificates.
* For ingress configurations allowing off-cluster access, you may need the ability to make DNS changes within your off-cluster environment.

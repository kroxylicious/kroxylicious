= Installing the Kroxylicious Operator

[role="_abstract"]
This section provides instructions for installing the Kroxylicious Operator.

ifndef::OpenShiftOnly[]
Install the proxy operator by applying the proxy installation files
endif::OpenShiftOnly[]

ifdef::OpenShiftOnly[]
Install the proxy operator using one of the following methods:

* By applying the proxy installation files
* From the OperatorHub in the OpenShift web console (Openshift Clusters only)
endif::OpenShiftOnly[]

Installation options and procedures are demonstrated using the example files included with Kroxylicious.

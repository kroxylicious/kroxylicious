= Glossary

[role="_abstract"]
Glossary of terms used in the SASL Inspection guide.

JWE:: JSON Web Encryption is an https://datatracker.ietf.org/doc/html/rfc7516[IETF standard] for exchanging encrypted data using JSON.
JWT:: JSON Web Token is an https://datatracker.ietf.org/doc/html/rfc8259[IETF standard] for securely transmitting information between parties as a JSON object.
JWS:: JSON Web Signature is an https://datatracker.ietf.org/doc/html/rfc7515[IETF-proposed standard] for signing arbitrary data.
Principal:: A principal is a component of a subject.
SASL:: Simple Authentication and Security Layer is an https://datatracker.ietf.org/doc/html/rfc4422[IETF framework] for handling authentication.
Subject:: A subject is the authenticated identity of a client.

= Configuring management HTTP endpoints

[role="_abstract"]
The proxy can run an HTTP server for exposing basic management information.
This is configured with the top level `management` property.

[id='con-configuring-admin-http-{context}']
.Configuration fragment showing the `management` property
[source,yaml]
----
# ... (filterDefinitions, virtualClusters etc)
management:
  bindAddress: 0.0.0.0
  port: 9190
  endpoints:
    prometheus: {}
----
where:

* `bindAddress` is the address the HTTP server should bind to. Defaults to `0.0.0.0`.
* `port` is the port the HTTP server should bind to. Defaults to `9190`.
* `endpoints` defines the services exposed by the management interface.
** `prometheus` If present and not null, exposes a Prometheus scrape endpoint at path `/metrics`.

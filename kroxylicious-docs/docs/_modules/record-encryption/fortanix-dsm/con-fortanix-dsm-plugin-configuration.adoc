= Fortanix DSM plugin configuration

[role="_abstract"]
For Fortanix DSM, the KMS configuration looks like this. Use the API key and Fortanix DSM Cluster URL values from the
KMS setup.

[source, yaml]
----
kms: FortanixDsmKmsService
kmsConfig:
  endpointUrl: <Fortanix DSM Cluster URL>
  apiKeySessionProvider:
    apiKey:
      passwordFile: /opt/fortanix-dsm/api-key
----
where:

* `kms` specifies the name of the KMS provider. Use `FortanixDsmKmsService`.
* `endpointUrl` provides xref:con-fortanix-dsm-setup-{context}[Fortanix DSM Cluster URL] including the protocol part, such as `https:` or `http:`.
* `passwordFile` is a reference to a file containing the API key.

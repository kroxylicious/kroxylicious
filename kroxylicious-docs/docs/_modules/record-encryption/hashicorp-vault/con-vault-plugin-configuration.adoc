= HashiCorp Vault plugin configuration

[role="_abstract"]
For HashiCorp Vault, the KMS configuration used by the filter looks like this.
Use the Vault Token and Vault Transit Engine URL values from the KMS setup.

[source, yaml]
----
kms: VaultKmsService
kmsConfig:
  vaultTransitEngineUrl: <vault transit engine service url>
  tls:
    # ...
  vaultToken:
    passwordFile: /opt/vault/token

----
where:

* `kms` specifies the name of the KMS provider. Use `VaultKmsService`.
* `kmsConfig` specifies configuration specific to the VaultKmsService.
** `vaultTransitEngineUrl` provides the  xref:con-vault-setup-{context}[Vault Transit Engine URL] including the protocol part, such as `https:` or `http:`.
** `tls` (Optional) TLS trust configuration.
** `vaultToken` references a file containing the Vault Token.

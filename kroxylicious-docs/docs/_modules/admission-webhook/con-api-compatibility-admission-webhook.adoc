= Custom resource API compatibility

[role="_abstract"]
The admission webhook uses a single custom resource: `KroxyliciousSidecarConfig`, in the `sidecar.kroxylicious.io` API group at version `v1alpha1`.

[WARNING]
====
The `v1alpha1` API version does not provide backwards-compatibility guarantees.
The schema, field names, semantics, and defaults may change between releases without a migration path.
Do not depend on the stability of this API for production workloads.
====
